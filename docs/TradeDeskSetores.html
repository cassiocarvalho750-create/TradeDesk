Configuração
Para o botão disparar o scan, informe seu repositório e um token do GitHub. Nada disso é enviado para lugar nenhum além da API do GitHub — fica salvo apenas neste navegador (localStorage).
O token precisa só de Actions: Read and write neste repositório. Veja o passo a passo no README_DEPLOY.
